# Sales Performance Dashboard | Tableau

# Overview

This project presents an interactive Sales Performance Dashboard built in Tableau using the Sample Superstore dataset. The dashboard enables business users to monitor sales performance, compare metrics across different regions, identify trends, and evaluate geographical sales distribution.

The objective was to transform raw transactional data into a user-friendly dashboard that supports faster and more informed business decisions.

# Business Problem

Businesses often struggle to monitor sales performance across multiple regions and identify areas requiring attention.

# This dashboard helps answer questions such as:

- Which region generates the highest sales?
- How are sales changing over time?
- Which locations contribute the most revenue?
- How do different sales metrics compare?
- Which regions need improvement?
